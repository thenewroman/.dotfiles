## Oh My Zsh
Install Oh My Zsh
```bash
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
## Stow
Use stow to create symbolic links
```bash
stow DIR
```
